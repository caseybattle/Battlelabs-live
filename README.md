# Battle Labs

Battle Labs is a conversion-focused landing page for async AI automation services. The current offer is positioned for coaches, consultants, creators, and digital product brands that want lead capture, follow-up, and lean funnel automation without calls.

## Local development

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```

## Build

```bash
npm run build
```

## Environment

Set `NEXT_PUBLIC_CONTACT_EMAIL` to the inbox that should receive Battle Labs inquiries. If omitted, the site falls back to `hello@battlelabs.live`.
