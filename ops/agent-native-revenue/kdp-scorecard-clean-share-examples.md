## Product Hunt comment replies (no-upvote, value-first)

**If someone asks “How do you use it?”**

```
My quick routine is: score the idea (1–5 each), note the weakest dimension, and only build the full book if the weakest dimension can be improved with a concrete change (narrower buyer, clearer occasion, simpler format).
It’s intentionally not a ranking predictor — it’s just a structured pause before building.
```

**If someone asks “What’s competition clarity?”**

```
Competition clarity is: can you describe a clear buyer + promise without cloning the top listings.
If you can’t write a 1-sentence “for X who want Y” without sounding generic, I treat that as low clarity.
```
