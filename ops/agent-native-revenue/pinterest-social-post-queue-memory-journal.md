## CTA rules (no-spend, platform-safe)

- Lead with a useful prompt or checklist; never lead with “buy”.
- Use soft CTAs: “save this”, “use this tonight”, “grab the free sample”, “see the preview”.
- Avoid guarantees: no “will sell”, “will rank”, “passive income”, “make $X”.
- If a community dislikes links, post the value first; add the link only if asked or in comments where allowed.
- One clear action per post (save OR sample OR preview), not all three.
