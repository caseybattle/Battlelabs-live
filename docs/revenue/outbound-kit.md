## Async Teardown Format
1. What is working right now
2. Where the likely conversion leak is
3. One change to make first
4. What I would build if we worked together
5. Soft CTA:
   If you want, I can scope this into a pilot build and keep the whole process async.

## Follow-Up Rules
- follow up after 48 hours
- follow up again after 5 days
- stop after 3 total touches unless they engage
- keep every follow-up shorter than the first message
