# Batch Review Playbook

## When to review
- after every 20 touches
- after every full batch
- immediately if a batch gets zero meaningful replies

## Review order
1. update the experiment log
2. fill out the touch review template
3. score the segment with the review rubric
4. decide where the next 20 touches go

## What counts as meaningful
- asks for the teardown
- asks what it would cost
- asks what specifically you would change
- asks for the proposal

## What does not count
- vague compliments
- “interesting” with no next step
- social engagement without commercial intent

## Kill rules
- zero positive replies after 20 touches: tighten the message
- zero positive replies after 40 touches: pause the segment
- repeated confusion about the offer: narrow the promise further

## Scale rules
- multiple teardown requests: keep feeding that segment
- proposal requests appearing: prioritize that segment next
- faster commercial replies from one segment: shift the next 20 touches there
