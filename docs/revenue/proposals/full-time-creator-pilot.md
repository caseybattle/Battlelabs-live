# Full Time Creator Pilot Build

## Goal
Create a lighter qualification layer before the heavier join flow so warmer creators can raise their hand without hitting the full commitment step immediately.

## Problem Summary
From the teardown, the main gap appears to be:
- the current process appears to move people into a relatively heavy join motion before a softer async step is visible

This matters because:
- strong interest can still be lost when the first step feels too committed
- creator-stage information is likely not being captured early enough
- follow-up cannot do enough work if the lead never enters the system

## Pilot Scope
This pilot build focuses on one specific outcome:
- add a pre-qualification layer before the heaviest join step

Typical components:
- softer pre-join CTA
- creator-stage sorting
- short async fit form
- follow-up sequence into the main join path
- async handoff notes

## Deliverables
- lighter pre-join entry path
- segmented qualification form
- follow-up sequence for each creator stage

## Timeline
- target turnaround: 72 hours to 5 days
- delivery model: async only

## Price
- pilot build fee: `$400`

## What is not included
- full application funnel rebuild
- brand-wide redesign
- backend membership platform overhaul
- ongoing support unless added separately

## Why this scope
The pilot is designed to prove whether a softer async step improves conversion into the existing high-authority join process.

## Close
If this scope looks right, the next step is a payment link and a simple async kickoff.
