# Creator Engines Pilot Build

## Goal
Tighten the path into the service so interested creators can be qualified asynchronously before the intro call becomes the only next step.

## Problem Summary
From the teardown, the main gap appears to be:
- the current page leans too heavily on the intro call to sort fit and intent

This matters because:
- warm but not-ready buyers may leave without entering any follow-up path
- the service likely attracts creators at different stages and urgency levels
- cleaner routing would reduce manual sorting before the call

## Pilot Scope
This pilot build focuses on one specific outcome:
- add a lighter async qualification and follow-up path before the intro call

Typical components:
- CTA cleanup
- segmented inquiry form
- shorter async fit questions
- follow-up sequence for not-yet-ready vs ready-now leads
- async handoff notes

## Deliverables
- softer pre-call inquiry path
- segmented intake form
- follow-up sequence tied to lead intent

## Timeline
- target turnaround: 72 hours to 5 days
- delivery model: async only

## Price
- pilot build fee: `$350`

## What is not included
- full site rebuild
- full CRM setup
- broad copy overhaul across the brand
- ongoing retainer support unless added separately

## Why this scope
This pilot isolates the highest-leverage front-end leak first: not enough async capture before the live call.

## Close
If this scope looks right, the next step is a payment link and a simple async kickoff.
