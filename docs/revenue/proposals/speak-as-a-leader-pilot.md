# Speak as a Leader Pilot Build

## Goal
Add a lighter async inquiry path before the consultation call so warm speaking and coaching leads can be captured before live scheduling.

## Problem Summary
From the teardown, the main gap appears to be:
- consultation calls are doing too much of the front-end qualification before a softer async step exists

This matters because:
- warmer leads may leave before booking live
- speaking and coaching inquiries likely need different sorting
- follow-up cannot do enough work if the lead never enters the system

## Pilot Scope
This pilot build focuses on one specific outcome:
- add segmented async inquiry and follow-up before the consultation step

Typical components:
- softer CTA before the consultation
- segmented inquiry for speaking vs coaching
- follow-up sequence for not-yet-ready leads
- simple routing for qualified leads

## Deliverables
- lighter pre-consultation inquiry path
- segmented form
- follow-up sequence for warmer leads

## Timeline
- target turnaround: 72 hours to 5 days
- delivery model: async only

## What is not included
- full site rebuild
- broad offer repositioning
- large automation stack
- ongoing support unless added separately

## Why this scope
This pilot is meant to prove whether a lighter async step improves capture before the consultation becomes the only next move.
